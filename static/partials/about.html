What is the CRISPR Library Designer?

The development of CRISPR (clustered regularly interspaced short palindromic repeats)-Cas has enabled new methods for interrogating gene function on a genome-wide scale. With just a simple alteration of the single-guide RNA (sgRNA) or direct repeat (DR) sequence, one can reprogram a nuclease to target different sites in the genome.

Recently, genome-scale CRISPR-Cas9 knockout libraries have become a popular method for performing loss-of-function screens. In 2015, our GeCKO library for large scale functional screens, lentiCRISPR V2 became the most requested plasmid on Addgene. This fantastic research tool has been requested from the repository nearly 1,000 times in 2015, yet we still received nearly daily requests for custom library constructs.

Therefore, we designed the CRISPR Library Designer: a semi-automated, tissue-specific software tool allowing scientists to design CRISPR knockout libraries as effortlessly as possible, without sacrificing control.

How does it work?

Technical Specifications

The software tool proceeds by blasting Ensembl transcripts for the PAM of the chosen enzyme, predicting on-target activity using machine learning models and weighting these results by mRNA expression of each exon (using RNA-seq profiles from the GTEx consortium and Gene Expression Omnibus). The user can individually reconfigure the targeted guides/exons and export the selected guides as a CSV spreadsheet, in a format suitable for upload to a custom oligo pool manufacturer, such as Twist Biosciences or CustomArray.

To promote scientific accuracy and development of this technology, we have open-sourced all code for the CRISPR Library Designer at the open-source repository GitHub.
